# ts-easing

Collection of easing function in TypeScript.

All functions accept a number in range of `[0..1]`.

All functions return a number, which is guaranteed to start at 0 and end at 1.

```
f(t), where t in [0..1]

f(0) -> 0
f(1) -> 1
```
